2. Apa perbedaan dari tag <p> dengan tag <br>, berikan penjelasannya!
Jawab: 
<br> yaitu untuk berpindah barisan ke barisan baru ketika disuatu paragraf di masukan kode ini,maka kalimat tersebut akan berpindah ke barisan baru
<p> yaitu memberikan awalan paragraf pada program

3. Apa perbedaan atribut title dan alt pada tag <img>, berikan penjelasannya!
Jawab: 
pada atribut title itu adalah seperti memberi caption di atas gambar tersebut

4. Untuk mengatur ukuran gambar, digunakan atribut width dan height. Agar tampilan gambar
proporsional sebaiknya kedua atribut tersebut diisi semua atau tidak? Berikan penjelasannya!
Jawab:
atribut width menentukan lebar gambar sedangkan atribut height untuk mengatur tinggi gambar
pada contoh diatas ukuran lebar (widht) sebesar 100 dan tinggi (height) sebesar 100

5. Pada link tambahkan atribut target dengan nilai atribut bervariasi ( _blank, _self, _top,
_parent ), apa yang terjadi pada masing-masing nilai antribut tersebut?
Jawab: 
Atribut TARGET digunakan untuk menetapkan di jendela atau di frame yg mana suatu dokumen yg dilink akan di tampilkan atau di muat di browser
Atribut TARGET tidak boleh digambarkan di dalam suatu dokumen HTML yg tipe deklarasinya adalah Strict.
Nilai atribut TARGET adalah _self, _blank, _top, dan framename
